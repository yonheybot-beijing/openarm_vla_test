## 操作步骤

## CPU:

#### 第一步：启动机械臂基础服务

打开终端，执行以下命令启动机械臂控制脚本：

运行

```bash
~/openarm_start_forward.sh
```

#### 第二步：启动遥操作程序

1. 新建终端，激活指定 conda 环境：

   ```bash
   conda activate tv1
   ```

2. 进入遥操作程序目录：

   ```bash
   cd ~/open_arm_teleperate_carmera-master/teleop
   ```

3. 运行遥操作控制脚本：

   ```bash
   python teleop_hand_and_arm.py
   ```

​		说明：本环境有可能缺少会2、3个包，若运行时报 `ModuleNotFoundError: No module named 'xxx'`，直接执行 `pip install xxx` 即可

#### 第三步：操作流程

1. 程序启动后，机械臂会自动回到**双臂举起的初始状态**，控制台提示 “按 r 继续”；

2. 待机械臂到达初始状态后，在控制台输入`r`并回车，进入**准备控制状态**；

3. **开始控制**：人站至能让双臂展开和身体完全纳入相机拍摄范围的位置（相机范围不要出现第二个人）后保持双臂与初始状态相似姿态，双手同时捏合**大拇指 + 小拇指**；

4. **结束控制**：双手同时捏合**大拇指 + 无名指**；

5. **夹爪控制**：捏合**大拇指 + 食指**控制夹爪张开 / 闭合。
